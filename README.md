## 💡 这是什么？

**小白老师**是一个面向儿童和青少年的性教育知识平台，通过 AI 智能问答和专业知识库，为孩子和家长提供科学、准确、温暖的性教育内容。

> 在中国，性教育长期缺位。我们希望用技术的力量，让每个孩子都能在安全、无压力的环境中了解自己的身体，学会保护自己。

## ✨ 核心功能

| 功能 | 描述 |
|------|------|
| 🤖 **智能问答** | 基于 RAG 技术的 AI 助手「小白」，用孩子听得懂的语言回答问题 |
| 📚 **知识百科** | 10 大主题，涵盖身体认知、青春期、情感关系、安全保护等 |
| 🎯 **年龄适配** | 根据不同年龄段，提供适宜深度的内容 |
| 🔒 **隐私安全** | 匿名使用，保护孩子隐私 |

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/AIB612/RAG-Chilren.git
cd RAG-Chilren

# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
npm start
```

## 🛠️ 技术栈

- **前端**: React 19 + TypeScript + TailwindCSS + Radix UI
- **后端**: Express 5 + Node.js
- **AI**: RAG (检索增强生成) 知识库
- **部署**: Render / Vercel / Docker
